# 🎬 MovieDex

MovieDex is a movie discovery web application built using HTML, CSS, and JavaScript. It uses the TMDB API to fetch real-time movie data, allowing users to browse trending, popular, top-rated, and upcoming movies, search for specific titles, and explore detailed information such as ratings, release dates, genres, cast, and trailers.

## Features

* 🔍 Search for movies by title
* 🎥 Browse Trending, Popular, Top Rated, and Upcoming movies
* 📄 View detailed movie information
* ⭐ Display ratings, release dates, genres, runtime, and overview
* 👥 View cast information
* ▶️ Watch official movie trailers
* 🔐 User authentication with Supabase

## Tech Stack

* HTML5
* CSS3
* JavaScript (ES6+)
* TMDB API
* Supabase

## Getting Started

1. Clone the repository.
2. Add your TMDB API key.
3. Configure your Supabase project.
4. Open `index.html` or run the project using Live Server.
